A* 経路探索（占有率0–100、block_threshold超えは障害物）

引数
occrows*cols 要素の占有率グリッド（行優先, y行x列, 0..100）
rows,colsグリッドサイズ
sx,systart (x,y) 左上原点, x=col, y=row
gx,gygoal (x,y)
block_thresholdこれ「より大きい」セルを障害物扱い（例: 50）
allow_diagonal0=4近傍, 非0=8近傍（斜めコスト=√2）
path_out呼び出し側確保バッファ（point_i32配列）。NULL可（長さだけ知りたいとき）
path_len_inout入力: path_outの最大要素数。出力: 実際に書き込んだ要素数。 start==goal の場合は 0（長さ0パス）。
errbufエラーメッセージ出力（NULL可）
errbuf_lenerrbuf のサイズ（bytes）。0可。
戻り値
plan_status_t 成功時 PLAN_OK。失敗時は各種ステータス（errbufに理由文字列）。

備考:

  • パスの並びは start→goal の順。
  • allow_diagonal=0 のときヒューリスティックはマンハッタン、!=0 のときはオクタイル。
  • スレッドセーフ（内部で静的状態を持たない）。
